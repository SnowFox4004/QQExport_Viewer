# QQExport_Viewer
A repositroy for viewing exported qq chatlogs / 用于查看导出的qq聊天记录的仓库

## 使用
1. 打开viewer.html
2. 选择聊天记录文件
3. 查看聊天记录

## 注意事项

1. 仅支持`QQNT_Export`仓库导出的`json`格式的**私聊**聊天记录
2. 仅用于临时查看，请勿用于其他用途
